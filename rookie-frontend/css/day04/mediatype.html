属性选择器

类选择器是对某一类元素设置样式 比如很多元素都可以设置相同的class，它们就属于同一类;伪类是某些元素在某些特殊状态下的样式，作用的同样是某个元素，只是加了一定的限制条件;
伪元素是对某些元素的特定内容(如第一行,第一个字)定义样式,操作的层级更深;属性选择器是当某个元素的属性满足一些条件时对它设置样式 其中判断条件有以下几种
= ~= |= ^= $= *=

计数器

记住一个原则 在遇到对应的选择器时,就会做出相应的counter操作,这样理解嵌套计数器就不难了

!important的使用

important属性是申明在一个样式表中的某个键值对之后的，每个键值对都可以设置;当某个元素同时设置了多个样式表,且这些样式表对同一个属性都做了定义,那么后面有important的键值对优先级最高
这也是为什么我们在使用important时要非常谨慎,一方面重复的important会让我们很困惑,到底哪一个才是最终的样式值,一方面他会使某些样式重叠后变得混乱,尽量不用或少用这个属性
